Date: Sat, 28 Jun 2026
Subject: Cologne Sanskrit Dictionaries update (June 2026)

Dear Sanskrit scholars, students, and dictionary enthusiasts,

After a three-year hiatus, the Cologne Digital Sanskrit Dictionaries newsletter is back.
Going forward it will be sent by email each month. To subscribe, visit
https://sanskrit-lexicon.github.io/csl-guides/users/newsletter

There is a great deal to catch up on.



- New documentation and guides site

The project now has a comprehensive documentation site at
https://sanskrit-lexicon.github.io/csl-guides/

It covers five areas: using the search interface and reading tools; deep-dive pages for
all 43 dictionaries in the catalog (with sample entries, abbreviation tables, and headword
keying conventions); the multi-dictionary comparison tool; a contributor correction
workflow; and a developer guide covering architecture, repositories, and the API.



- Compare dictionaries side by side

The new multi-dictionary tool lets you look up one Sanskrit word across Monier-Williams,
Apte, Böhtlingk-Roth, and Grassmann simultaneously, with live entries fetched from the
CDSL API. Each column resolves the headword per dictionary — useful for seeing how the
same lemma is treated across scholarly traditions.

https://sanskrit-lexicon.github.io/csl-guides/tools/multi-dictionary



- Reading Monier-Williams — a step-by-step guide

A new guide covers MW's four layers of alphabetical order, its symbols, and how to trace
a word to its dhātu. Sixty-nine self-check questions each link to a live CDSL entry, so
every quiz doubles as a cross-check between the print edition and the digital text.

https://sanskrit-lexicon.github.io/csl-guides/users/reading-monier-williams



- Abbreviations standardized across all 43 dictionaries

Each dictionary deep page now embeds a full abbreviation table sourced from the
authoritative list for that dictionary. A cross-dictionary comparison makes it easy to see
how the same abbreviation is used (or clashes) across dictionaries.

https://sanskrit-lexicon.github.io/csl-guides/dictionaries/abbreviations-comparison



- Headword keying conventions documented

Each dictionary page now includes a section on its headword keying conventions — how the
digital record's search key relates to the form printed in the dictionary. This matters
particularly when using the API or working with raw data files.



- Prefaces for Grassmann and Macdonell

The prefaces for Grassmann (Wörterbuch zum Rig-Veda) and Macdonell (A Practical Sanskrit
Dictionary) are now accessible online, bringing the total to seven dictionaries with full
preface text available.

https://sanskrit-lexicon.github.io/csl-guides/dictionaries/ocr-prefaces



Corrections and feedback are always welcome through the GitHub repository:
https://github.com/sanskrit-lexicon/csl-guides
